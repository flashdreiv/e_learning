# E-Learning System

**Tech Stack:**

_Backend:_
 - Django
 - Django Rest Framework

_Frontend_
- React
- Redux
- Redux-thunk
- Zustand
- Semantic UI


# Getting Started

_Backend_ 
- cd backend
- pip install -r requirements.txt
- python manage.py runserver

_Frontend_
- cd frontend
- npm install
- npm start
